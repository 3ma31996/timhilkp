Panduan Konversi PDF ke Excel

Step-by-step menggunakan iLovePDF untuk data statistik

1
Membuka Website iLovePDF
  1. Buka browser (Chrome/Edge/Firefox)
  2. Ketik kata kunci "ilovepdf" pada kolom pencarian Google
  3. Pilih hasil pertama bertuliskan "iLovePDF | Online PDF tools for PDF lovers"
  4. Klik link tersebut untuk masuk ke website iLovePDF
2
Masuk ke Fitur PDF to Excel
  1. Setelah halaman utama iLovePDF terbuka, scroll hingga bagian menu tools
  2. Cari kotak warna hijau dengan ikon Excel
  3. Klik tombol "PDF to Excel" di bagian kanan bawah grid tools
  4. Pastikan tool yang dipilih adalah PDF to Excel, bukan PDF to Word/JPG
3
Upload File PDF
  1. Pada halaman Convert PDF to EXCEL, klik tombol merah "Select PDF file"
  2. Alternatif: tarik file PDF ke area abu-abu bertuliskan "or drop PDF here"
  3. Tunggu sampai jendela pemilihan file muncul
  4. Pilih file PDF yang ingin dikonversi

üí° Tips:

Pastikan file PDF tidak terproteksi password dan berisi data tabel yang jelas

4
Konfigurasi Layout Hasil

Pengaturan Layout:

  • Multiple sheets ‚Üí jika PDF memiliki beberapa tabel/halaman yang berbeda
  • One sheet ‚Üí jika hanya 1 tabel besar
  • Untuk dataset long / multi-tabel per halaman, pilih Multiple sheets
5
Proses Konversi & Download
  1. Klik tombol merah "Convert to EXCEL" untuk mulai proses konversi
  2. Tunggu hingga proses selesai (biasanya <30 detik)
  3. Jangan tutup browser selama proses berlangsung
  4. Klik tombol merah "Download EXCEL" setelah proses selesai
  5. Simpan file ke folder tujuan
6
Mengecek struktur tabel di dalam Excel
  1. Buka file Excel hasil konversi.
  2. Periksa apakah header tabel (Kolom A‚ÄìD) sudah sesuai dengan tabel di PDF
  3. Perhatikan jika terdapat tanda ‚Äú*‚Äù atau titik (.) sebagai missing value
  4. Catat adanya multi-header (judul baris 1 dan baris 2) untuk proses cleaning selanjutnya
7
Bandingkan Hasil konversi Excel dengan tabel asli
  1. Buka kembali PDF sumber
  2. Cocokkan judul tabel, nama kolom, dan urutan variabel
  3. Pastikan angka dan kode industri sesuai
  4. Identifikasi simbol khusus seperti * atau bold (ini biasanya footnote atau missing)
8
Struktur dataset standar sebelum input
  1. Buka file Excel baru
  2. Buat header dataset dalam format data: Kode Industri,Tanah, GedungMesin dan Perlengkapan, KendaraanPerangkat Lunak / Database, Modal Tetap Lainnya, Total
  3. Tambahkan merge cell / merge column di template ini
9
Struktur tabel yang di-convert sesuai format asli
  1. Periksa setiap kolom apakah sudah sesuai nama variabel: Tanah, Gedung, Mesin dan Perlengkapan, Kendaraan, Perangkat Lunak/Database, Modal Tetap Lainnya, Total
  2. Pastikan tidak ada kolom yang hilang akibat proses konversi
  3. Cek apakah format angka masih ada pemisah ribuan (titik), jika ya ‚Üí tidak perlu dihapus dulu
  4. Pastikan baris kosong atau tanda ‚Äú-‚Äù menandakan missing value, bukan 0
10
Identifikasi data abnormal/missing yang tidak rapi
  1. Scroll ke tengah tabel dan cari blok data yang tidak beraturan
  2. Perhatikan nilai yang ditandai * pada kolom tertentu ‚Üí ini menunjukkan data tidak tersedia di publikasi
  3. Catat kode industri yang memiliki baris kosong berturut-turut
  4. Tandai range data tersebut untuk proses pembersihan manual atau imputasi nanti
  5. Dan karena tidak banyak, perbaiki secara manual dengan masukan 2 baris lalu isi sesuai file asli (.pdf)
11
Tambah Kolom Metadata
  1. Tambahkan kolom ‚ÄúKeterangan‚Äù di sebelah kanan kolom Kode Industri
  2. Isi dengan nilai IB atau IBS sesuai kategori pada tabel PDF
  3. Data ini penting untuk filter pemrosesan statistik berdasarkan ukuran industri
  4. Pastikan tidak ada sel kosong di kolom metadata
12
Struktur Sheet Kompilasi
  1. Gunakan nama sheet standard: COMPILE_PENAMBAHAN untuk data penambahan/pembelian
  2. Hindari mencampur data jenis lain (misal pengurangan, depresiasi) dalam sheet ini
  3. Buat sheet terpisah seperti COMPILE_PENGURANGAN jika dataset lain tersedia
  4. Simpan tabel awal setelah input sekitar 10‚Äì20 baris sebagai checkpoint
13
Simpan Workbook agar struktur tidak hilang
  1. Klik menu File di kiri atas Excel
  2. Pilih Save As untuk menyimpan file baru
  3. Jangan gunakan Save, karena file masih kosong atau auto-generated
14
Penentuan Folder tujuan yang konsisten
  1. Klik Browse agar bisa memilih direktori manual
  2. Hindari penyimpanan di lokasi acak seperti Desktop
  3. Gunakan folder proyek/data master yang konsisten agar mudah diakses
15
Penamaan file
  1. Gunakan format nama file: [Tahun]_[Sumber]_[Topik]_[Kategori].xlsxContoh:2018_BPS_Statistik Industri Manufaktur Industri_IB.xlsx
  2. Pastikan tidak menggunakan spasi di awal/akhir atau karakter ilegal seperti / * |.
  3. Klik Save
16
Data tahun yang lain
  1. Lakukan hal yang sama terhadap data tahun-tahun yang lain
  2. Kumpulkan dalam 1 folder
  3. File siap di compile
17
Menyalin Data dari Workbook Sumber
  1. Buka salah satu workbook tahun (contoh: 2022)
  2. Pilih data dari baris pertama hingga baris terakhir tabel (jangan pilih kolom kosong)
  3. Pastikan seluruh kolom: Kode Industri s.d Total termasuk dalam area seleksi
  4. Tekan Ctrl + C untuk menyalin
18
Template Database Final
  1. Buat workbook baru dengan struktur kolom berikut:Kode Industri, Tipe Industri, Tanah, GedungMesin dan Perlengkapan, Kendaraan, Perangkat Lunak/Database, Modal Tetap Lainnya, Total, Tahun, Jenis, Digit
  2. Jangan gunakan merge cell
  3. Kolom metadata harus berada di kanan blok variabel utama
19
Paste ke Template
  1. Klik sel A2 pada sheet database
  2. Paste data menggunakan:Paste Special ‚Üí Values (tanpa format)
  3. Pastikan baris header tidak ikut ter-copy
  4. Verifikasi apakah data masuk sesuai kolom
20
Penambahan Metadata Tahun & Jenis
  1. Pada kolom Tahun, isi nilai sesuai dokumen (contoh: 2022)
  2. Pada kolom Jenis, isi sesuai tabel: Pembelian/Penambahan dan Pembuatan/Perbaikan Besar
  3. Jangan mempersingkat deskripsi karena konsistensi penting untuk model data
  4. Pada kolom Digit, gunakan formula: =LEN(A2)
  5. Drag ke bawah hingga seluruh baris terisi
  6. Digit akan menjadi metadata untuk filtrasi analisis sektoral: 2-digit = sector, 3-digit = subsector, 4-digit = industri detail
21
Kompilasi Multi-Tahun dalam 1 Sheet
  1. Salin data tahun berikutnya (2021,2020, 2019, dst)
  2. Paste tepat di bawah baris terakhir data sebelumnya
  3. Pastikan kolom metadata terisi konsisten pada tiap baris
  4. Sheet diberi nama:2018‚Äì2022
22
Menyimpan database multi-tahun pertama
  1. Klik menu File ‚Üí Save As
  2. Pilih lokasi penyimpanan database jangka panjang (bukan folder sumber tahunan)
  3. Pilih Browse
23
Penamaan database gabungan
  1. Gunakan format: [RentangTahun]_[Sumber]_[Topik]_[Kategori]_db_v01.xlsx, Contoh:2018-2022_BPS_Statistik Industri Manufaktur Industri_db_v01.xlsx
  2. Huruf kecil/huruf besar tidak penting, yang penting konsistensi
  3. Klik Save
  4. Konversi dari file pdf menjadi excel selesai

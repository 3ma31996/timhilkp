Anatomi Do-file Stata yang Baik

üìù Struktur Do-file untuk Export ke Excel

/*============================================== PROJECT: Analisis Data Survey 2023 AUTHOR: Tim Hilirisasi 3 - Bappenas DATE: $S_DATE PURPOSE: Clean data & export to Excel ==============================================*/ // 1. SETUP ENVIRONMENT clear all set more off set mem 500m cd "C:\Project\Data\" cap log close log using "analysis_$S_DATE.log", replace // 2. LOAD DATA use "raw/survey.dta", clear // 3. DATA CLEANING drop if missing(age) replace income = . if income < 0 gen ln_income = log(income) label variable ln_income "Log of Income" // 4. ANALYSIS summarize age income education regress income age education // 5. EXPORT TO EXCEL export excel using "output.xlsx", firstrow(variables) replace // 6. FINISH log close di "Analysis complete. Output saved as output.xlsx"

Metode 1: export excel (Paling Mudah)

üì§ Ekspor Dataset Lengkap ke Excel

üìù Basic Syntax:
export excel using "filename.xlsx", firstrow(variables) replace
// BASIC EXPORT - SEMUA DATA export excel using "data_output.xlsx", firstrow(variables) replace // EXPORT DENGAN PILIHAN KOLOM export excel id age income education using "subset.xlsx", /// firstrow(variables) sheet("Survey_Data") replace // EXPORT KE SHEET TERTENTU DAN POSISI export excel using "results.xlsx", /// firstrow(variables) sheet("Data") cell(A5) replace // EXPORT TANPA LABEL VALUE (hanya kode) export excel using "data_nolabel.xlsx", firstrow(variables) nolabel replace // EXPORT MULTIPLE SHEETS export excel using "full_report.xlsx", firstrow(variables) sheet("Data") replace export excel region sales using "full_report.xlsx", /// firstrow(variables) sheet("Summary") replace

‚ö° Tips export excel:

1. Gunakan firstrow(variables) untuk header
2. Selalu pakai replace untuk overwrite file lama
3. Specify sheet name untuk organisasi yang rapi
4. Untuk value labels, gunakan decode() sebelum export
5. Untuk missing values, handle di Stata sebelum export

üîß Pre-Export Checklist:

  1. Check data types - numeric vs string
  2. Handle missing values - . menjadi blank atau 0
  3. Convert labeled variables - gunakan decode()
  4. Format dates - Stata dates ‚Üí Excel dates
  5. Remove unnecessary variables - kurangi file size
  6. Sort data jika perlu - untuk presentasi rapi

Metode 2: esttab untuk Regression Tables

üìà Ekspor Hasil Regresi Profesional

üì¶ Install Package:
ssc install estout, replace
// INSTALL PACKAGE (jika belum) ssc install estout, replace // RUN REGRESSIONS regress income age education estimates store model1 regress income age education experience estimates store model2 // EXPORT KE EXCEL - BASIC esttab model1 model2 using "regression_results.xlsx", replace // EXPORT DENGAN FORMATTING LENGKAP esttab model1 model2 using "regression_full.xlsx", replace /// b(3) se(3) /// // 3 decimal places star(* 0.1 ** 0.05 *** 0.01) /// // Significance stars r2 ar2 /// // R-squared stats scalar(N F) /// // N and F-stat title("Regression Results") /// // Title mtitles("Model 1" "Model 2") // Model titles // EXPORT UNTUK PUBLIKASI esttab model1 model2 using "pub_ready.xlsx", replace /// cells(b(star fmt(3)) se(par fmt(3))) /// stats(N r2_a, fmt(0 3) labels("Observations" "Adjusted R-squared")) /// varwidth(20) /// title("Table 1: Regression Analysis")

üéØ Key Options

  • b(3) - 3 decimal coefficients
  • se(3) - 3 decimal std errors
  • star() - significance stars
  • r2 ar2 - R-squared stats
  • scalar(N) - observations
  • title() - table title

üìä Output Types

  • .xlsx - Excel format
  • .rtf - Word format
  • .tex - LaTeX format
  • .csv - CSV format
  • .html - Web format

// EXAMPLE 1: SUMMARY STATISTICS TABLE summarize age income education, detail matrix summ = r(StatTotal)' putexcel set "summary_stats.xlsx", replace putexcel A1 = "Summary Statistics" putexcel A2 = matrix(summ), names putexcel A1, bold border(bottom) putexcel A2:C10, nformat(number_d2) putexcel save // EXAMPLE 2: REGRESSION RESULTS WITH FORMATTING regress income age education matrix b = e(b) matrix V = e(V) putexcel set "regression_formatted.xlsx", replace putexcel A1 = "Variable" B1 = "Coefficient" C1 = "Std. Error" D1 = "t-stat" putexcel A2 = matrix(b)', transpose putexcel C2 = matrix(sqrt(diag(V)))', transpose putexcel D2 = formula(C2/B2) putexcel A1:D1, bold border(bottom) putexcel B2:D10, nformat(number_0.000) putexcel save // EXAMPLE 3: MULTIPLE SHEETS putexcel set "full_report.xlsx", sheet("Data") replace putexcel A1 = matrix(data_matrix) putexcel set "full_report.xlsx", sheet("Results") modify putexcel A1 = matrix(results_matrix) putexcel save

üì• Cara Import .dta dan .xlsx ke Excel

üìä Metode 1: Direct Import .xlsx

Jika sudah export dari Stata:

  1. Double-click file .xlsx
  2. Excel langsung terbuka
  3. Data sudah dalam format Excel
  4. Siap untuk analysis lebih lanjut

‚ÜîÔ∏è Metode 2: Import .dta ke Excel

Menggunakan Power Query:

  1. Excel ‚Üí Data tab ‚Üí Get Data
  2. From File ‚Üí From Stata (.dta)
  3. Butuh: "Stata file reader" add-in
  4. Power Query import dengan labels
  5. Transform jika perlu

üìÅ Metode 3: CSV Intermediate

Jika tidak bisa direct:

  1. Stata: outsheet using "data.csv"
  2. Excel: Data ‚Üí From Text/CSV
  3. Pilih file CSV
  4. Transform di Power Query
  5. Save as .xlsx

üîß Power Query untuk Import Stata Output:

Issue Symptom Power Query Solution
Missing values "." muncul di Excel Replace "." with null
Date formats Numbers bukan dates Add custom column: Date + number
Label codes 1,2 bukan Male,Female Conditional column: if 1 then "Male"
Scientific notation 1.23E+10 Change data type to text

Cleaning Data Stata di Excel

üßπ Handle Common Stata Export Issues

‚ö†Ô∏è Masalah Umum Stata ‚Üí Excel:

‚Ä¢ Missing values ‚Üí "." muncul sebagai teks
‚Ä¢ Label values ‚Üí hanya kode numeric (1,2)
‚Ä¢ Date values ‚Üí Stata date numbers
‚Ä¢ String whitespace ‚Üí spasi di awal/akhir
‚Ä¢ Scientific notation ‚Üí untuk angka besar

// EXCEL FORMULAS UNTUK CLEANING: // 1. HANDLE MISSING VALUES (Stata's ".") =IF(A2=".", "", A2) // 2. CONVERT STATA DATE TO EXCEL DATE // Stata: days since 01jan1960 =DATE(1960,1,1) + A2 // Kemudian format sebagai date // 3. CONVERT LABEL CODES TO TEXT =IF(B2=1, "Male", IF(B2=2, "Female", "Unknown")) // 4. TRIM WHITESPACE =TRIM(A2) // 5. FIX SCIENTIFIC NOTATION =TEXT(A2, "0") // 6. HANDLE SPECIAL MISSING (.a, .b, .c) =IF(OR(A2=".a", A2=".b", A2=".c"), "Missing", A2)

üîß Cleaning di Stata (Better)

  • decode gender, gen(gender_str)
  • format date %td
  • mvdecode _all, mv(.)
  • destring, replace
  • tostring, replace format()

üìù Cleaning di Excel

  • Find & Replace "." ‚Üí blank
  • Text to Columns
  • Flash Fill untuk patterns

Troubleshooting Common Issues

üîß Solusi Masalah Ekspor Stata

‚ùå Common Errors & Solutions:

Error Cause Solution
"export excel not found" Stata version < 13 Use outsheet or upgrade
"file already exists" File not replaced Add replace option
Labels not exported export excel doesn't export labels Use decode() before export
Excel file corrupt Export interrupted Use CSV intermediate
"esttab not found" Package not installed ssc install estout
Date format wrong Stata dates are numeric Convert in Stata or Excel

üîç Debugging Tips

  • Check Stata version: version
  • Test with small dataset first
  • Export to CSV as backup
  • Use log to track issues
  • Verify data before export

üîÑ Alternative Methods

  • Use outsheet for CSV
  • Copy-paste from Results window
  • Use list with output capture
  • Export matrices with matwrite
  • Use third-party tools (Stat/Transfer)

üìÅ File Management

  • Backup original .dta files
  • Use version control in filenames
  • Organize by date/project
  • Document all export steps
  • Keep log files for audit
